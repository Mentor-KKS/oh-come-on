# OH COME ON!

**Ein Rage-Platformer, der dich anlügt.** / **A rage platformer that lies to you.**

> *"That spike was always there."*

---

## Was ist das? / What is this?

**DE:** OH COME ON! ist ein brutaler 2D-Platformer, bei dem nichts so ist wie es scheint. Vertraue nichts und niemandem — schon gar nicht dem Spiel selbst. Du wirst sterben. Oft. Viel Spass.

**EN:** OH COME ON! is a brutal 2D platformer where nothing is what it seems. Trust nothing and no one — especially not the game itself. You will die. A lot. Have fun.

---

## Spielen / Play

**https://mentor-kks.github.io/oh-come-on/**

Einfach im Browser starten — kein Download nötig. / Just open in your browser — no download required.

---

## Features

- **18 Level** voller Fallen, Tricks und Gemeinheiten / full of traps, tricks, and cruelty
- **Multiplayer** — Erstelle Räume und race gegen Freunde (2-4 Spieler) / Create rooms and race against friends (2-4 players)
- **Speedrun-Modus** — Im Singleplayer und Multiplayer, mit Bestzeiten / In singleplayer and multiplayer, with best times
- **Community Editor** — Baue eigene Level und teile sie per Code / Build your own levels and share them via code
- **Shared Levels** — Spiele Level von anderen direkt im Spiel / Play levels from others directly in-game
- **Phasen-System** — Level 1-15, 16-30 + Hardcore Speedrun

## Multiplayer

1. Im Hauptmenü **MULTIPLAYER** wählen / Select **MULTIPLAYER** in the main menu
2. Name und Farbe einstellen / Set name and color
3. **Raum erstellen** oder einem Raum mit **5-stelligem Code** beitreten / **Create room** or join with a **5-digit code**
4. Modus wählen: Single Level, Phase, All Levels, Speedrun Phase, Speedrun All oder Shared Level / Choose mode
5. Alle auf Ready — Host startet das Rennen / Everyone ready — host starts the race

---

## Community Editor

Im Hauptmenü unter **EDITOR** erreichbar. / Accessible via **EDITOR** in the main menu.

- Plattformen, Spikes und Fallen platzieren / Place platforms, spikes, and traps
- Level testen und exportieren / Test and export levels
- Level-Code teilen — andere können ihn über **Shared Level** spielen / Share level code — others can play it via **Shared Level**
