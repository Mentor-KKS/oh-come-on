Store exported community-editor test levels here.

Recommended workflow:

1. Build and test a level in `community-editor.html`.
2. Use `Export JS Level File`.
3. Save the downloaded file into `test-levels/`.
4. When a test level is ready for the main game, move or adapt it into `js/levels/`.

These files are useful as backups, prototypes, and review builds for other testers.
